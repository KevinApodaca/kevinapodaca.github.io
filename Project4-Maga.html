MAGA (A modern database system)

An advising system for the Computer Science department at the University of Texas at El Paso. Allows users to create advising appointments, create school schedules, edit advising records, and more through a web interface. A semester-long project built for the CS4342 Database Management course as taught in Fall 2019 at UTEP.
